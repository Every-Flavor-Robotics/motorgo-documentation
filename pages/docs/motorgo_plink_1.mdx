## Features

* Replace this with a pinout diagram with the core information about what the different things on the pins do
* ESP32-S3 with Wifi and BT connectivity
* Form factor of a PiHat (we do not currently have the EEPROM requirements of the PiHat Spec)
* DC barrel jack and on-board buck from 6-17V into 5.1V up to 6A (enough to power pi4 and pi5)
* 9 dof IMU (LSM6DSOTR-C accel and optional gyro + LIS3MDLTR mag)
* USB-C programming compatible with arduino IDE
* Qwiic port x2 for easy integration with sensors
* Full gpio breakout on 40Pin header + full RPi GPIO passthrough if using as a hat
* 4 bidirectional DC motor channels up to 2A each channel
* 4 Encoder ports for use with EncoderGo 5pin jst for close loop feedback on each motor

## Setup
* Create a split panel:
    * Left side: Standalone
    * Right side: PiHat
    *
The MotorGo Plink can be used as a standalone motor controller or as a PiHat. In standalone mode, the onboard ESP32-S3 is programmable via PlatformIO or the Arduino IDE. You can program the standalone Plink in the Arduino Programming Language.

In PiHat mode, the MotorGo Plink can be attached to any Raspberry Pi with a 40 pin header. You can use our Python API to command the MotorGo Plink from your Raspberry Pi, upgrading an already powerful computer into an affordable full-feature robotics platform.
