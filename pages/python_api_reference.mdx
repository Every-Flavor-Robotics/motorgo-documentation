# Python API Reference

<Callout type="info" emoji="ðŸ¤–">
    Applies to Plink, Axis
</Callout>

The MotorGo Python API allows you to control the PiHat-compatible MotorGo boards on your Raspberry Pi. To set up your MotorGo and Raspberry Pi, follow the steps in the [quickstart guide](/pihat_mode/quickstart).

To use the MotorGo Python API, first import your MotorGo board from the `motorgo` module:

```python filename="python"
from motorgo import Plink # or Axis
from motorgo import ControlMode # Used for congfiguring the control mode for the motors
```

The `Plink` and `Axis` classes are the main interfaces for controlling the MotorGo Plink and Axis boards, respectively. Jump to the [Plink API reference](/python_api_reference/plink) or [Axis API reference](/python_api_reference/axis).
