# Python API Guide

This guide will walk you through using all of the features of the MotorGo Python API.

- [Getting Started](/python_api_reference/getting_started): Import the MotorGo board, configure the power supply voltage, and connect to the board.
- [Reading Encoder Data](/python_api_reference/encoder): Read the encoder data from the MotorGo board.
- [Motor Control](/python_api_reference/motor_control): Control the motors on the MotorGo board.
- [Reading IMU Data](/python_api_reference/imu): Read the IMU data from the MotorGo board.
- [Reading QWIIC Sensors](/python_api_reference/battery): Read data from connected QWIIC sensors.

Note: The Python API currently only supports the MotorGo Plink board. Support for the MotorGo Axis board is coming soon!
